Panduan Aplikasi

Buka Halaman Prediksi

Akses Website: Buka halaman web prediksi gagal jantung melalui browser Anda.

Klik Disini

Upload Dataset

Form Upload Dataset: Temukan form untuk upload dataset pada halaman.

Upload File: Klik tombol “Choose File” atau “Browse” dan pilih file dataset gagal jantung dalam format .xlsx atau .csv dari komputer Anda.

Submit Dataset: Setelah memilih file, klik tombol “Upload” untuk mengunggah dataset ke server.

Pilih Tipe Data dan Kelas Label

Tentukan Kolom yang Tidak Digunakan: Pilih kolom-kolom yang tidak akan digunakan dalam analisis. Kolom ini mungkin termasuk ID unik, nama pasien, atau informasi lainnya yang tidak relevan untuk prediksi.

Tentukan Tipe Data: Pilih atribut yang akan digunakan sebagai kelas label (target) untuk prediksi.

Set Kelas Label: Klik tombol “Choose File” atau “Browse” dan pilih file dataset gagal jantung dalam format .xlsx atau .csv dari komputer Anda.

Atur Parameter Model Decision Tree

Max Depth:

  • Definisi: Batas maksimum kedalaman pohon keputusan.
  • Fungsi: Membatasi kedalaman pohon untuk mencegah overfitting.

Min Samples Split:

  • Definisi: Jumlah minimum sampel yang diperlukan untuk membagi node internal.
  • Fungsi: Memastikan bahwa node tidak dibagi jika tidak ada cukup data, untuk menghindari pembagian yang tidak perlu.

Min Samples Leaf:

  • Definisi: Jumlah minimum sampel yang harus ada di leaf node.
  • Fungsi: Mencegah pembentukan leaf node dengan terlalu sedikit sampel yang dapat menyebabkan overfitting.

Tampilkan Hasil Model

Proses Model: Setelah parameter diatur, klik tombol “Run” atau “Train Model” untuk memulai proses pembuatan model klasifikasi menggunakan Decision Tree.

Tampilkan Hasil: Sistem akan menampilkan hasil evaluasi model seperti akurasi, precision, recall, dan support.

Masukkan Data untuk Prediksi

Form Input Data: Temukan form untuk memasukkan data sesuai kondisi Anda.

Isi Data: Masukkan nilai untuk setiap atribut yang diminta sesuai dengan kondisi kesehatan Anda.

Hasil Prediksi

Submit Data: Klik tombol “Prediksi" setelah mengisi data.

Tampilkan Hasil: Sistem akan menampilkan hasil prediksi yang menyatakan apakah Anda terkena gagal jantung atau tidak.
